# Product Catalog App

## Deskripsi

Product Catalog App adalah aplikasi React yang menggunakan Axios untuk mengambil data produk dari Fake Store API. Aplikasi ini menampilkan daftar produk, kategori produk, pencarian produk, filter, sorting, dan detail produk dalam modal.

## Teknologi yang Digunakan

- React JS
- Axios
- Fake Store API
- JavaScript
- CSS Inline Style

## Cara Menjalankan Aplikasi

1. Clone repository

```bash
git clone https://github.com/USERNAME/NAMA_REPOSITORY.git
```

2. Masuk ke folder project

```bash
cd nama-project
```

3. Install dependency

```bash
npm install
```

4. Jalankan aplikasi

```bash
npm run dev
```

5. Buka browser

```text
http://localhost:5173
```

## API yang Digunakan

Fake Store API

```text
https://fakestoreapi.com/products
https://fakestoreapi.com/products/categories
```

## Fitur Utama

### 1. Product List
Menampilkan seluruh produk dari Fake Store API.

### 2. Loading Indicator
Menampilkan pesan loading saat data sedang diambil dari server.

### 3. Error Handling
Menampilkan pesan error apabila terjadi kegagalan request API.

### 4. Search Produk
Mencari produk berdasarkan judul secara real-time.

### 5. Category Filter
Memfilter produk berdasarkan kategori yang diambil langsung dari API.

### 6. Product Detail Modal
Menampilkan informasi detail produk dalam popup modal.

### 7. Responsive Grid
Tampilan produk menyesuaikan ukuran layar.

## Fitur Tambahan

### 1. Sorting Harga
- Termurah ke Termahal
- Termahal ke Termurah

### 2. Rating Filter
Filter produk berdasarkan rating.

### 3. Price Range Filter
Filter berdasarkan harga minimum dan maksimum.

### 4. Pagination
Menampilkan produk per halaman.

### 5. Add To Cart
Menambahkan produk ke keranjang.

### 6. Toast Notification
Menampilkan notifikasi saat produk berhasil ditambahkan ke cart.
